# Asksuite test (frontend)
Este é um projeto frontend desenvolvido com Vue.js, utilizando o framework Quasar para componentes de UI.


## Tecnologias Utilizadas

- **Vue 3** como framework;
- **Quasar** como biblioteca de componentes;
- **Pinia** para gerenciamento de estado;
- **Javascript**
# Pré-requisitos
```
- Node.js 20+
- PostgreSQL (Rodando API e front local)

```
## Configuração do Ambiente
1. Clone o repositório:
```
git clone https://github.com/ricrdev-eng/ask-test-frontend cd ask-test-frontend
```

2. Instale as dependências:
```
 npm install
```

3. Configure as variáveis de ambiente:
   Crie um arquivo `.env` na raiz do projeto com as seguintes variáveis:
4. Caso queira rodar o front com o backend local **altere** o **NODE_ENV** para **DEV** 

```
API_URL=https://ask-test-production.up.railway.app
NODE_ENV=production
```
## Instale as dependências

```bash
npm install
```

### # Iniciar servidor de desenvolvimento

```bash
npm run dev
```

### Build the app for production

```bash
npm run build
```
